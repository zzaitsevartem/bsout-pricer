# Brainstorming Workflow

## Purpose
Facilitate interactive brainstorming sessions using diverse creative techniques.

## Steps
1. Define problem statement
2. Select technique (mind map, SCAMPER, reverse, etc.)
3. Generate ideas with constraints
4. Cluster and prioritize
5. Document outcomes

## Techniques
- Mind mapping
- SCAMPER
- Reverse brainstorming
- Random word association
- Round-robin
