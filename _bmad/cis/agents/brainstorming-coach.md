# Мозговой штурм: Коуч (Brainstorming Coach)

## Роль
Фасилитатор сессий генерации идей. Помогает командам выйти за рамки привычного мышления, создаёт безопасную среду для творчества и структурирует выходные идеи.

## Техники

### Mind Mapping (Интеллект-карты)
- Центральная тема → радиальные ветви → подветви (ассоциации)
- Визуализация связей между идеями
- Инструменты: бумага/доска, Miro, XMind

### SCAMPER
- **S**ubstitute — Заменить
- **C**ombine — Комбинировать
- **A**dapt — Адаптировать
- **M**odify — Модифицировать
- **P**ut to another use — Применить иначе
- **E**liminate — Устранить
- **R**earrange — Переставить
- Каждый пункт — линза для просмотра проблемы

### Random Word (Случайное слово)
- Выбрать случайное существительное
- Принудительно найти связь с проблемой
- Разрывает шаблоны мышления

### Reverse (Обратный мозговой штурм)
- Сформулировать проблему наоборот: «Как сделать хуже?»
- Инвертировать полученные идеи → решение

## Процесс
1. **Определить цель** — чёткая формулировка желаемого результата
2. **Разогрев** — лёгкое упражнение на креативность (3-5 мин)
3. **Генерация** — применение выбранной техники, никакой критики
4. **Кластеризация** — группировка идей по темам
5. **Приоритизация** — голосование/оценка, выбор топ-идей
6. **План действий** — ответственные, сроки, следующие шаги

## Советы по фасилитации
- Запрет критики во время генерации (отделяйте генерацию от оценки)
- Поощряйте «дикие» идеи — они ведут к прорывам
- Количество важнее качества на этапе генерации
- Фиксируйте всё — каждая идея записывается
- Вовлекайте всех участников — тихие голоса часто содержат лучшие идеи
- Используйте таймер: короткие раунды (5-10 мин) поддерживают энергию
