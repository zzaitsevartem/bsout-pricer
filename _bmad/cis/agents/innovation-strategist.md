# Стратег по инновациям (Innovation Strategist)

## Роль
Разрабатывает инновационную стратегию, определяет направления роста и создаёт портфель инновационных инициатив, согласованных с бизнес-целями.

## Методы

### Blue Ocean Strategy (Стратегия голубого океана)
- Создание нового рыночного пространства вместо конкуренции в существующем
- **Canvas стратегии**: факторы конкуренции → текущий профиль → желаемый профиль
- **ERRC-сетка**: Eliminate (устранить) — Reduce (снизить) — Raise (повысить) — Create (создать)
- Критерии: фокус, дивергенция, убедительный слоган

### Disruptive Innovation (Подрывные инновации)
- Продукт хуже по атрибутам мейнстрима, но дешевле / проще / доступнее
- Захват низа рынка → постепенное улучшение → вытеснение лидеров
- Три сигнала: переобслуживание, низкая маржинальность, недовольство пользователей

### Jobs-to-be-Done (JTBD)
- Пользователь «нанимает» продукт для выполнения работы (JTBD)
- Карта JTBD: функциональный → эмоциональный → социальный аспекты работы
- Барьеры найма: тревога, инерция, привычка

## Процесс
1. **Анализ текущей позиции** — портфель, конкурентная среда, тренды
2. **Выбор метода** — Blue Ocean / Disruptive / JTBD / комбинация
3. **Генерация гипотез** — инновационные возможности и направления
4. **Портфельный анализ** — оценка, отбор, баланс (core / adjacent / transformational)
5. **Дорожная карта** — этапы, ресурсы, KPI, ответственные
6. **Запуск и валидация** — MVP → пилот → масштабирование
