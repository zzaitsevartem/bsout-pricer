# Workflow: Мозговой штурм (Brainstorming)

## Purpose
Проведение структурированной сессии генерации идей, от постановки задачи до плана действий.

## Input
- Описание проблемы или возможности
- Опционально: контекст, ограничения, критерии успеха
- Участники (3-10 человек)

## Steps

### Step 1: Define Goal (Определить цель)
- Действия: сформулировать 1-2 чётких вопроса для штурма; записать на видном месте
- Формат: «Как мы могли бы ...?»
- Критерий: вопрос понятен всем участникам одинаково
- Длительность: 5-10 мин

### Step 2: Warm-Up (Разогрев)
- Действия: лёгкое упражнение на креативность (история из трёх слов, необычное применение предмета)
- Цель: снять барьеры, настроить на генерацию
- Длительность: 3-5 мин

### Step 3: Generate (Генерация)
- Действия: применить технику (mind map, SCAMPER, random word); фиксировать все идеи
- Правила: никакой критики, количество > качество, развивайте чужие идеи
- Длительность: 15-30 мин (раундами по 5-10 мин)

### Step 4: Cluster (Кластеризация)
- Действия: сгруппировать похожие идеи; дать названия кластерам
- Инструменты: стикеры, Miro, доска
- Длительность: 10-15 мин

### Step 5: Prioritize (Приоритизация)
- Действия: оценить идеи по матрице (эффект × сложность); выбрать топ-3-5
- Метод: голосование точками (dot voting) или шкала 1-5
- Длительность: 10-15 мин

### Step 6: Action Plan (План действий)
- Действия: для каждой приоритетной идеи — ответственный, первый шаг, срок
- Результат: документ с решениями и трекер выполнения
- Длительность: 10-15 мин

## Output
- Список всех сгенерированных идей (кластеризованный)
- Топ-приоритетные идеи с обоснованием
- План действий: ответственные, сроки, следующие шаги
