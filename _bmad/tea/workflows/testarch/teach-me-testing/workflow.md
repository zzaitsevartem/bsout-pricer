# Teach Me Testing Workflow

## Purpose
Teach testing progressively through structured educational sessions.

## Steps
1. **Assess level** — determine current knowledge and goals
2. **Choose topic** — select testing area (unit, integration, e2e, NFR)
3. **Explain concept** — present theory with real examples
4. **Hands-on practice** — guided exercise with code
5. **Review** — discuss results and clarify questions
6. **Next steps** — recommend follow-up topics

## Input
- Learner's current skill level
- Desired learning outcomes

## Output
- Practical understanding of testing topic
- Code examples and exercises
