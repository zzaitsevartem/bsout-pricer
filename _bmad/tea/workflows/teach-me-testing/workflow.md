# Workflow: Обучение тестированию (Teach Me Testing)

## Purpose
Обучить команду основам тестирования: методология, инструменты, практики. Формирование тестовой культуры в проекте.

## Input
- Состав участников (роли, текущий уровень)
- Цели обучения (базовый / продвинутый / экспертный)
- Пилотный проект или модуль для практики

## Steps

### Step 1: Assess (Оценка)
- Действия: опрос / интервью участников (текущие знания, инструменты, ожидания)
- Определить уровни: Junior, Middle, Senior в контексте тестирования
- Результат: профиль группы, план обучения

### Step 2: Design Program (Дизайн программы)
- **Модуль 1**: Основы тест-дизайна (эквивалентность, границы, таблицы решений)
- **Модуль 2**: Виды тестирования (unit, integration, e2e, NFR)
- **Модуль 3**: ATDD / BDD — примеры, Given-When-Then
- **Модуль 4**: Автоматизация (выбор фреймворка, написание тестов)
- **Модуль 5**: CI/CD — интеграция, отчёты, метрики
- Результат: программа обучения с длительностью модулей

### Step 3: Hands-On Workshop (Практика)
- Действия: разобрать реальный модуль проекта — написать тесты вместе
- Формат: mob programming или парное программирование
- Каждый участник пишет минимум 1 тест каждого типа
- Результат: написанные тесты и разобранные кейсы

### Step 4: Code Review Session (Разбор тестов)
- Действия: общий ревью написанных тестов, обсуждение улучшений
- Темы: читаемость, покрытие, мокание, изоляция
- Результат: улучшенные тесты, усвоенные практики

### Step 5: Pair Testing Sessions (Парное тестирование)
- Действия: опытный тестировщик + участник проводят тест-сессию
- Фокус: ручное исследовательское тестирование + автоматизация
- Результат: найденные баги, закреплённые навыки

### Step 6: Retrospective & Next Steps (Ретроспектива)
- Действия: собрать обратную связь, оценить прогресс
- Определить план развития каждого участника
- Результат: отчёт об обучении, план дальнейшего роста

## Output
- Обученная команда с базовыми навыками тестирования
- Набор тестов, написанных в ходе обучения
- Материалы курса: презентации, чек-листы, шпаргалки
- План индивидуального развития участников
