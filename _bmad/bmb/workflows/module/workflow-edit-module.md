# Workflow: Редактирование модуля

**Purpose:** Внести изменения в существующий BMAD-модуль.

## Steps

1. **Assess current**
   - Прочитать README модуля, манифест, список агентов и воркфлоу
   - Зафиксировать текущее состояние (git status)

2. **Plan changes**
   - Определить, что добавляется / изменяется / удаляется
   - Оценить влияние на соседние модули и манифесты
   - Согласовать план изменений

3. **Implement**
   - Внести изменения в файлы агентов и/или воркфлоу
   - Обновить манифест модуля (agent-manifest.csv)
   - Обновить README модуля

4. **Validate**
   - Проверить целостность структуры модуля
   - Проверить ссылки внутри модуля и на внешние модули
   - Убедиться, что удалённые файлы не имеют внешних зависимостей

## Input
- Описание требуемых изменений
- Текущее состояние модуля

## Output
- Обновлённые файлы модуля
- Обновлённый манифест
- Отчёт о валидации
