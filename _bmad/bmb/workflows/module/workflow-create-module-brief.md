# Workflow: Краткое создание модуля

**Purpose:** Быстро сформировать BMAD-модуль по краткому описанию.

## Steps

1. **Define purpose**
   - Записать одну строку: для чего модуль

2. **Define scope**
   - Определить границы: что входит, что НЕ входит в модуль
   - Перечислить зависимости от других модулей

3. **Define agents needed**
   - Составить список агентов (роль + краткое описание)
   - Для каждого указать триггер активации

4. **Define workflows needed**
   - Составить список воркфлоу (название + цель)
   - Для каждого указать, какие агенты задействованы

5. **Define config**
   - Определить, нужны ли конфигурационные файлы (YAML/JSON)
   - Если да — описать структуру конфига

## Input
- Краткое описание модуля (1-3 предложения)

## Output
- План создания модуля
- Список агентов и воркфлоу
- (Опционально) Структура конфига
