# Workflow: Создание модуля

**Purpose:** Создать новый BMAD-модуль «с нуля».

## Steps

1. **Define scope**
   - Определить purpose модуля, границы, критерии успеха
   - Согласовать с Tech Lead

2. **Design structure**
   - Спроектировать файловую структуру: `agents/`, `workflows/`, `config/`
   - Определить, какие агенты и воркфлоу войдут в модуль

3. **Create agents**
   - Для каждого агента: создать .md с секциями Role, Process, Standards, Checklist
   - Зарегистрировать агентов в манифесте модуля

4. **Create workflows**
   - Для каждого воркфлоу: создать .md с Purpose, Steps, Input, Output
   - Убедиться, что шаги ссылаются на существующих агентов

5. **Validate**
   - Проверить структуру: все ли каталоги и файлы на месте
   - Проверить ссылки: нет ли битых перекрёстных ссылок
   - Проверить compliance стандартам BMAD

## Input
- Бизнес-требования к модулю
- Архитектурный контекст

## Output
- Файловая структура модуля
- Набор агентов и воркфлоу
- Зарегистрированный модуль в манифесте
