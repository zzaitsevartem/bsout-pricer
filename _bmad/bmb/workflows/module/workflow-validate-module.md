# Workflow: Валидация модуля

**Purpose:** Проверить корректность и полноту BMAD-модуля.

## Steps

1. **Check structure**
   - Проверить наличие обязательных каталогов: `agents/`, `workflows/`
   - Проверить наличие README.md в корне модуля
   - Проверить соответствие naming convention

2. **Check links**
   - Проверить все внутренние ссылки в файлах агентов и воркфлоу
   - Проверить ссылки на манифесты (_bmad/_config/)
   - Убедиться, что нет битых cross-модульных ссылок

3. **Check compliance**
   - Каждый агент содержит: Role, Process, Standards, Checklist
   - Каждый воркфлоу содержит: Purpose, Steps, Input, Output
   - Все агенты зарегистрированы в agent-manifest.csv

4. **Report**
   - Сформировать отчёт: результаты по каждому пункту
   - Указать найденные проблемы и рекомендации
   - Присвоить статус: OK / Warnings / Errors

## Input
- Путь к модулю

## Output
- Отчёт валидации (статус + детали)
