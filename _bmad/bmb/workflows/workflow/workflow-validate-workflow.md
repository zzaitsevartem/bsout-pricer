# Workflow: Валидация воркфлоу

**Purpose:** Проверить корректность BMAD-воркфлоу.

## Steps

1. **Check structure**
   - Проверить наличие Purpose, Steps, Input, Output
   - Проверить нумерацию шагов (1., 2., 3. …)
   - Каждый шаг — одно действие одного агента

2. **Check completeness**
   - Все ли шаги имеют исполнителя (агента)
   - Все ли агенты существуют в манифесте
   - Input первого шага достижим, Output последнего — полезен

3. **Check references**
   - Все ссылки на агентов и другие воркфлоу корректны
   - Нет циклических зависимостей между воркфлоу

4. **Report**
   - Сформировать статус: OK / Warnings / Errors
   - Описать проблемы (если есть)

## Input
- Файл воркфлоу (.md)
- Манифест модуля

## Output
- Отчёт валидации воркфлоу
