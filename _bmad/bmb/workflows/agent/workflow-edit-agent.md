# Workflow: Редактирование агента

**Purpose:** Внести изменения в существующего BMAD-агента.

## Steps

1. **Review current**
   - Прочитать текущий файл агента
   - Проверить манифест модуля на наличие записи об агенте
   - Найти все ссылки на агента из других файлов

2. **Plan changes**
   - Определить, какие секции требуют изменений
   - Оценить влияние изменения на контракты выхода
   - Если breaking change — уведомить зависимые воркфлоу

3. **Implement**
   - Внести изменения в файл агента
   - Обновить запись в agent-manifest.csv (при необходимости)
   - Обновить связанные воркфлоу (при изменении контракта)

4. **Validate**
   - Проверить, что файл агента корректен
   - Проверить, что все ссылки на агента работают
   - Убедиться, что изменения совместимы с существующими воркфлоу

## Input
- ID агента
- Описание изменений

## Output
- Обновлённый файл агента
- Обновлённые зависимые файлы
