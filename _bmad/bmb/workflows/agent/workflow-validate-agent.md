# Workflow: Валидация агента

**Purpose:** Проверить корректность и полноту BMAD-агента.

## Steps

1. **Check structure**
   - Проверить наличие всех обязательных секций: Role, Process, Standards, Checklist
   - Проверить форматирование (валидный Markdown)

2. **Check activation**
   - Убедиться, что способ активации явно указан
   - Если событийный — проверить, что тип события задокументирован
   - Если ручной — проверить, что описаны ожидаемые входные данные

3. **Check completeness**
   - Проверить, что все пункты Checklist реализуемы
   - Проверить, что Process не содержит разрывов (нет шага без исполнителя)
   - Проверить, что Input/Output консистентны

4. **Report**
   - Сформировать статус: OK / Warnings / Errors
   - Перечислить найденные проблемы

## Input
- Файл агента (.md)
- Манифест модуля

## Output
- Отчёт валидации агента
