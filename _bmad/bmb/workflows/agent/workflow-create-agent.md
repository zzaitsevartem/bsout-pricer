# Workflow: Создание агента

**Purpose:** Создать нового BMAD-агента.

## Steps

1. **Define role**
   - Сформулировать название и область ответственности агента
   - Определить, в каком модуле будет размещён агент

2. **Write persona**
   - Описать роль, процесс работы, стандарты, чеклист
   - Указать формат выходных данных (контракт выхода)

3. **Define activation**
   - Определить триггеры активации (ручной вызов / событие / шедул)
   - Если событийный — указать тип события

4. **Validate**
   - Проверить: все обязательные секции присутствуют
   - Проверить: ссылки на другие агенты/воркфлоу корректны
   - Зарегистрировать агента в agent-manifest.csv модуля

## Input
- Описание требуемой роли агента

## Output
- Файл агента (.md)
- Запись в agent-manifest.csv
